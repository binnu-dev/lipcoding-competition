# Backend for Mentor-Mentee Matching App

## Stack
- Node.js (LTS 22.x)
- Express.js
- SQLite (file-based)
- JWT for authentication

## Structure
- /backend
  - /src
    - controllers/
    - models/
    - routes/
    - middlewares/
    - utils/
    - app.js
    - db.js
  - package.json
  - README.md

## API Docs
- Swagger UI: http://localhost:8080/swagger-ui
- OpenAPI: http://localhost:8080/v3/api-docs

## Run
```bash
cd backend
npm install
npm start
```
