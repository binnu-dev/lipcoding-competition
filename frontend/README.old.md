# Frontend for Mentor-Mentee Matching App

## Stack
- React (with Vite or Create React App)
- Axios (API requests)
- Tailwind CSS (optional, for fast styling)

## Structure
- /frontend
  - /src
    - components/
    - pages/
    - App.js
    - index.js
  - public/
  - package.json
  - README.md

## Run
```bash
cd frontend
npm install
npm start
```

## Default URL
- http://localhost:3000
